# Customer Sales Data Analysis Using Python

## Project Overview
This project analyzes customer sales data using Python. 
The goal of the project is to understand sales performance, product trends, regional performance, and customer payment behavior.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Dataset
The dataset contains 30,000 rows of sales records with the following fields:

OrderID  
Product  
Category  
Price  
Quantity  
TotalSales  
Region  
City  
PaymentMethod  
OrderDate  

## Analysis Performed

1 Regional Sales Analysis  
Total sales were calculated for each region to identify which region generates the most revenue.

2 Product Sales Analysis  
Products were grouped to determine which products contribute the highest sales.

3 Category Sales Analysis  
Sales were analyzed based on product categories.

4 Payment Method Analysis  
Customer payment preferences were analyzed.

5 Monthly Sales Trend  
Sales were analyzed by month to understand sales trends.

## Key Insights

- South region generated the highest revenue.
- West region had slightly lower sales compared to other regions.
- Some products contribute significantly to total revenue.
- Digital payment methods like UPI and Credit Card are frequently used.
- Monthly sales trends show variations across different months.
